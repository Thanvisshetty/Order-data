# Order-data
As part of my internship at Global Quest Technologies, I developed an interactive Order Tracking Dashboard to analyze sales, profit, customer behavior, and regional performance.
Under the guidance of my reporting head, I worked on transforming raw business data into actionable insights that can support strategic decision-making for leadership.
This dashboard enables management to:

 ✅ Identify high-performing states
 
 ✅ Evaluate profitable customer segments
 
 ✅ Analyze operational efficiency
 
 ✅ Support data-driven business strategies
 
Tools Used: Power BI | Excel | Python (EDA) | SQL | Data Visualization

# Kids Screen time
I am excited to share my Power BI dashboard project on Kids Screen Time Analysis.
This project focuses on understanding children’s digital habits by analyzing screen usage patterns, sleep duration, physical activity, device usage, and parental control impact across different countries and age groups.
Through this project, I applied data cleaning, visualization, and dashboard design techniques to convert raw data into meaningful insights that can help in understanding behavioral trends.
